YouTube Channel Grouper

Create channel groups (music, sports, news, etc.) and see the latest video from each group on your YouTube homepage.

YouTube API key

To fetch the latest video for each channel, this extension uses the YouTube Data API. Create an API key in the Google Cloud Console and paste it here. This key is stored only in your browser (locally).

If you leave this empty, groups will still work, but the homepage section cannot show latest videos.

Groups

Manage your channel groups. You can also create and assign groups directly on a channel page.

Deleting a group will unassign it from any channels that used it.

Danger Zone

This will erase all groups, channel assignments, and your API key. This action cannot be undone.
